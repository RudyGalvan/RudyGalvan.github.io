# RudyGalvan.github.io

Personal site: 8-bit styled portfolio for GitHub Pages.

## Local preview

From this directory:

```bash
python3 -m http.server 8000
```

Open `http://localhost:8000`.

## GitHub Pages

1. Push `main` (or your default branch) to GitHub.
2. Repository **Settings → Pages**.
3. **Build and deployment**: Source **Deploy from a branch**, branch **main**, folder **/ (root)**.
4. The site will be available at `https://rudygalvan.github.io` (may take a minute after the first deploy).

Edit copy and links in `index.html`. Colors and layout live in `css/style.css`.
